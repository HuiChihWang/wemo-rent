## Description
This project handling the backend of the wemo renting service.

## Project setup
```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests (prepare .env.test file)
$ npm run test:e2e
```
